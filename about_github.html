# Github相关 ## 1.弄一个库,上传一些书上的作业代码 ###### 具体操作：新建库,之后github会弹出提示页面,里面包含你在任何系统上的继续操作的方法。git add README.md, add你的文档就可以了。 ## 2.git做笔记 ###### 如果想要笔记好看的话,学习markdowm好了。很容易掌握的语法规则,网上也有不少在线编辑的模拟器。 ## 3.代码与本地同步 ###### add, commit, push三步,你的代码就同步到远端了.远端有修改的话,pull就是本地与远端合并. ## 4.制作自己的主页 ###### 学习一丢丢Jekyll,利用github生成自己的个人博客。不用太会,因为你可以fork自一些大方的分享者.然后持续更新你的blog,也能多点亮很多天.
